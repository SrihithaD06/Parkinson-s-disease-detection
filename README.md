# 🧠 Parkinson’s Disease Detection using Machine Learning

## 📌 Overview
This project aims to detect Parkinson’s disease using biomedical voice features and machine learning techniques. The model classifies whether a person is affected by Parkinson’s or is healthy based on extracted feature data.

## 🎯 Objective
To build a machine learning model that can accurately classify Parkinson’s disease using feature-based input data.

## 🧪 Methodology
- Loaded and processed the dataset
- Removed unnecessary columns
- Split data into training and testing sets
- Applied feature scaling using StandardScaler
- Trained a Support Vector Machine (SVM) model
- Evaluated performance using accuracy, classification report, and confusion matrix

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 📊 Results
The model was successfully trained and evaluated using SVM. It provides good accuracy in classifying Parkinson’s and healthy individuals.

## 📁 Dataset
- Parkinson’s dataset with biomedical voice measurements
- Features include frequency, amplitude, and other signal-based characteristics
